# Веб-галерея изображений на Flask

Курсовая работа по разработке веб-приложения для управления галереей изображений.

## 📋 Описание проекта
Веб-приложение позволяет:
- Автоматически сканировать папки с изображениями
- Загружать новые изображения (JPG, PNG, GIF)
- Управлять избранными изображениями
- Просматривать галерею на любых устройствах
